Grant

  • Grant is authentication middleware for Express https://github.com/simov/grant
  • All providers use a test application called Purest
  • Purest is a REST API client library https://github.com/simov/purest which you can use for subsequent calls to these providers
  • Currently this example server uses statically defined example scope options
  • The generated access tokens are not stored anywhere
  • At any point go to your provider's account settings and revoke the access for the Purest app
